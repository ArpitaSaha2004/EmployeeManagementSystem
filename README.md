# Employee Management System

## Overview
Employee Management System is a web-based application developed using Java technologies to manage employee records efficiently.

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat

## Features
- Employee Login
- Add Employee
- View Employees
- Update Employee Details
- Delete Employee
- Employee Dashboard
- Payslip Generation
- Database Connectivity using JDBC

## Project Structure
- src/ - Java source files
- web/ - JSP pages and web resources
- lib/ - Required libraries
- EmployeeManagement.sql - Database script
- EmployeeManagementSystem.war - Deployable WAR file

## Database
1. Create a MySQL database.
2. Import `EmployeeManagement.sql`.
3. Update database credentials in src/com/ems/util/DBConnection.java

## Deployment
1. Import project into IntelliJ IDEA or Eclipse.
2. Configure Apache Tomcat.
3. Deploy the WAR file or run the project from the IDE.
4. Access the application through the browser.
