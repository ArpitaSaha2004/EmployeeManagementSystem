# Employee Management System

## Overview

Employee Management System is a web-based application developed using Java technologies to manage employee records, authentication, salary details, and payslip generation efficiently.

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* MySQL
* Apache Tomcat
* JavaMail API
* Render
* Aiven MySQL

## Features

* Admin Login
* Employee Login
* Add Employee
* View Employees
* Update Employee Details
* Delete Employee Records
* Employee Dashboard
* Payslip Generation
* Email Payslip Sending
* Database Connectivity using JDBC
* Session Management

## Project Structure

* `src/` - Java source files (Servlets, DAO, Models, Utilities)
* `web/` - JSP pages and web resources
* `lib/` - External libraries and dependencies
* `EmployeeManagement.sql` - Database script
* `EmployeeManagementSystem.war` - Deployable WAR file

## Database

**Database:** Aiven MySQL Cloud Database

**Tables:**

* `admin`
* `employees`
